Project Introduction

This Rocket Motor Test Stand (RMTS) was designed and built by RMTS Industrial and is intended for use by the Student Rocket Program (SPR). The contents of this manual conform to the standards required by the National Association of Rocketry (NAR) and are designed strictly in accordance with the following project documents: System Requirements Specification (SRS) and Technical Requirements Document (TRD).

The RMTS is a modular rocket engine test platform compatible with NAR 1/8A to NAR I class engines, able to acquire and store test data. The core of the system comprises two sections: RMTS Control and Data Collection Unit and RMTS Structural and Thrust Measurement Unit. The RMTS is built for all-weather operations in Alaska, the test stand can be disassembled and packed into three transport containers, allowing for rapid deployment by a single person within 30 minutes.
